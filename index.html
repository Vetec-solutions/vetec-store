Software

Tres formas de operar la misma tecnología.

VETEC distribuye la familia de software de Alisys Robotics para que cada cliente pueda elegir el nivel de control que necesita: desde operación simple desde Android hasta supervisión multi-activo o dashboards ejecutivos configurables.

Plataforma Robótica Modular

Visión ejecutiva y operación configurable

Dashboards, widgets ilimitados, programación de misiones y una capa agnóstica pensada para supervisión y control adaptados a cada despliegue.

Arcus Lite

Control simple desde Android

Una experiencia directa para operar el robot desde móvil o tablet, ideal para uso 1 a 1, demostraciones, pilotos y una entrada sencilla a la robótica.

Arcus Defense

Mapa central y gestión avanzada

Entorno orientado a vigilancia y operación táctica con monitorización centralizada, vídeo y teleoperación sobre mapa.
